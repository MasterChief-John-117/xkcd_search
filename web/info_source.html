xkcd_search

A simple, quick way to search xkcd comics


What does it do?

  • Search by title
  • Search by all metadata
  • Search by community generated tags
  • Add a tag

Search by title

Endpoint: GET /title/:query


Search by all metadata

Endpoint: GET /search/:query


Search by tags

Endpoint: GET /tags/:query


Add a tag

Endpoint: GET /tags/:num/add/:tag
